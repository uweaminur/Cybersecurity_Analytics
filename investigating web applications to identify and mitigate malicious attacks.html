Step-2: Meaning of each error code ¶

From above list, i get only 4 error code and from this error code, I find below explananion for each code,¶

  • Error code 200 = The request was received and understood and is being processed.
  • Error code 404 = The page you were trying to reach on a website couldn't be found on their server
  • Error code 301 = Moved Permanently is used for permanent redirecting.
  • Error code 401 = The client request has not been completed because it lacks valid authentication credentials for the requested resource.

Above explanation of the codes, I understand that only one code explained the user who trying to access to the server is not the real user and who do not know the credentials. If the user knows the credential, then there will not be generated such code more than thousands time. Again, if the user is real and sometimes mistakenly input the wrong credential (which is natural) the error code would generates more than thousands time which is suspicious.
